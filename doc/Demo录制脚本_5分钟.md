# 稳盈 / Poise · 5 分钟 demo 录制脚本

> 给录屏者：按场次顺序操作，旁白可读出 `🎙️` 后的文字，或用 AI 配音。
> 总时长 **5:00 ± 15s**，建议帧率 30fps、分辨率 1920×1080，导出 MP4 H.264。
> 关键时刻按 `📸` 提示停顿 2-3 秒便于剪辑。

---

## 录制前准备（10 分钟）

```bash
# 1. 全栈启动
cd ~/CodeProjects/Poise-CFLS
make up
docker compose exec api alembic upgrade head
docker compose exec api python -m poise.scripts.bootstrap
make seed

# 2. 浏览器准备
# 打开 Chrome 隐身窗口（避免插件干扰），分辨率 1920×1080
# 关闭书签栏、清空 zoom（Cmd+0）
# 准备 4 个标签页（预加载）：
#   - http://localhost:3000/login
#   - http://localhost:3000/dashboard
#   - http://localhost:3000/forecast
#   - http://localhost:3000/chat

# 3. 终端准备
# 字号放大到 16+，深色主题
# 准备好命令：bash scripts/smoke.sh （备用 B-roll）

# 4. 屏幕录制软件
# Mac：QuickTime / OBS Studio / ScreenFlow
# 录系统音 + 麦克风
```

## 2:00 - 3:00  三档 MILP 求解 + what-if（60s）

**🎬 画面**：左侧导航 → **方案对比**

操作序列：
1. 点击「**重新求解（三档 MILP）**」
2. **📸 截屏点**：约 100ms 后出现三档卡片
   - 稳健 ¥570K · 投资 16 笔 · 0 融资 · 合规 ✓
   - 折中 ¥722K · 投资 15 笔 · 0 融资 · 合规 ✓
   - 进取 ¥779K · 投资 13 笔 · 0 融资 · 合规 ✓
3. 鼠标依次悬停三张卡，让观众看清单调递增
4. **📸 截屏点**：「安全垫曲线对比」图（三档曲线叠加）
5. 滚到下方折中档 PlanAction 明细表，鼠标停 2 秒
6. 点击右上角「🧪 **what-if 沙盘**」打开抽屉
7. 已经填好 W3 锁 ¥80M 并购款（快捷场景）
8. 点击「应用锁定 + 重算三档」
9. 抽屉显示「完成 (73ms) · conservative: ¥503K | balanced: ¥662K | aggressive: ¥718K」

🎙️ **旁白**：

> "求解三档方案 —— 90 毫秒。稳健、折中、进取，收益从 ¥57 万到 ¥78 万单调递增。"
>
> "每档都满足 6 类硬约束加 3 类精修：流动性安全、授信上限、集中度上限、活钱比例下限……"
>
> "what-if 沙盘 —— 假设我必须锁定 W3 的 ¥80M 并购款，重算。70 毫秒，三档全部更新。收益下降但仍然全部可行，这就是参谋的价值。"

## 3:00 - 4:15  对话参谋（75s）

**🎬 画面**：左侧导航 → **对话参谋**

操作序列：
1. 点击「+ 新会话」
2. 输入框打字：**"未来 13 周哪几周会紧？"**（边打边录，1 秒）
3. 回车发送
4. **📸 截屏点**：流式过程
   - 看到「调用 run_forecast ✓」Badge 出现
   - 然后「🧠 推理中...」实时显示中文 token 流
   - 最后正文 markdown 渲染（"W11 安全垫仅 ¥4.5M..."）
5. 等 LLM 答完，**📸 截屏点**最终回复（约 10-15s）
6. 输入第二条：**"假设北辰电气回款延后 2 周，重算方案"**
7. 回车发送
8. 看到连环 tool 调用：query_position → apply_overrides → build_and_solve → explain_plan
9. **📸 截屏点**：折叠的工具返回 JSON，展开一个看真实数据
10. 最终回复包含具体新方案

🎙️ **旁白**：

> "这是对话参谋。我问：未来 13 周哪几周会紧？"
>
> "它先调 run_forecast 拿到真实预测，然后开始推理 —— 注意所有金额都来自后端真实求解，绝不凭空生成，这是数字纪律。"
>
> "再追问：假设北辰电气回款延后 2 周。它自动连环调用 4 个工具：解析假设 → 修改输入 → 重新求解 → 生成解释。从问句到完整答案，全程对话即可，不用我去手动改假设。"
